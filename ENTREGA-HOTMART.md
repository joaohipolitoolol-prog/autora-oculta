# Entregável Autora Oculta — como entregar na Hotmart

## O que é

Área digital completa do método (HTML), no mesmo visual do funil:

**URL de produção:** https://autora-oculta.vercel.app/entregavel/

Pasta no repo: `public/entregavel/`

## Conteúdo incluído

1. Início / como usar  
2. Tu proyecto (salva título, seudónimo, premisa no navegador)  
3. Método Autora Oculta  
4. Mapa do mercado oscuro  
5. Creador de universo + prompts  
6. Estrutura de até 25 capítulos  
7. Personagens  
8. Biblioteca de prompts (com substituição automática do projeto)  
9. Seudônimo e identidade visual  
10. Sinopsis e descrição comercial  
11. Guía de publicación  
12. Modelos de anuncios  
13. Projeto exemplo  
14. Plan 7 días  

## Como configurar na Hotmart

1. Abra o produto **Autora Oculta** na Hotmart.  
2. Em **Conteúdo / Entrega**, adicione um conteúdo do tipo **Link** ou **Área de membros** com:

   `https://autora-oculta.vercel.app/entregavel/`

3. No e-mail de compra, confirme que o comprador recebe esse link.  
4. (Recomendado) No texto de boas-vindas Hotmart:

   > Accede a tu método aquí. En la página de resultado del test, toca “Copiar mi resultado” y pégalo en “Tu proyecto”.

## Segurança

O link é **semi-público** (quem tiver a URL acessa). Isso é comum em produtos Hotmart baratos. Para trancar de verdade no futuro: Hotmart Club / login / members area nativa.

Não coloque o link nos anúncios — só pós-compra.

## Fluxo ideal compradora

1. Faz o quiz no funil  
2. Vê o resultado → compra  
3. Hotmart envia o link `/entregavel/`  
4. Cola o resultado em **Tu proyecto**  
5. Segue Plan 7 días + prompts  

## Atualizar o conteúdo

Edite os arquivos em `public/entregavel/`, faça commit/push ou `vercel --prod`. A URL não muda.
