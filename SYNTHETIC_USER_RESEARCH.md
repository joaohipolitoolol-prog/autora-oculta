# SYNTHETIC_USER_RESEARCH — Autora Oculta

**Importante:** esto es investigación **sintética** con personas inventadas para diseño.  
No es feedback real de compradoras. No usar como prueba social.

## Personas (12) — hallazgos recurrentes

| Persona | Dolor principal | Mejora aplicada en producto |
|---------|-----------------|-----------------------------|
| Nunca escribió | No sabe el siguiente paso | Card “Tu próxima acción” |
| Fanfic | Quiere capítulos ya | Módulo Capítulos first-class |
| Renta extra | Espera milagro | Copy sin promesas de ingresos |
| Miedo a aparecer | Privacidad | Identidad + ayuda seudónimo |
| Solo móvil | Menú eterno | Bottom nav 5 ítems |
| Internet lenta | Carga / offline | PWA + datos locales |
| Abandona largo | Overwhelm | Plan 7 + pasos cortos |
| Quiere automático | “haz el libro” | Aviso: IA externa, 1 prompt = 1 escena |
| Experta en IA | Prompts flojos | Biblioteca ampliada + Prompt Maestro |
| Baja digital | No encuentra instalar | Guía Instalar + iOS |
| Ya tiene historia | Importar | Import quiz + backup |
| Quiere serie | Continuidad | Prompt serie + continuidad |

## Tareas sintéticas (usabilidad)
Importar quiz, próxima acción, crear protagonista, copiar prompt, pegar en capítulo, exportar, instalar PWA, borrar datos — todas tienen ruta visible en la IA actual.
